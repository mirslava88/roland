# Таймер доклада

## Описание
Таймер обратного отсчёта для контроля времени доклада с отображением на внешнем дисплее.

## Функционал

### Управление (панель управления)
- Установка времени доклада (часы:минуты)
- Кнопки быстрого добавления: +1 мин, +5 мин, +10 мин
- Ввод произвольного количества минут
- Запуск / пауза / сброс таймера
- Выбор звукового файла для сигнала окончания времени
- Выбор звукового файла для предупреждения за 1 минуту

### Отображение (внешний дисплей)
- Оверлей поверх презентации в правом нижнем углу
- Настраиваемое расположение и масштаб
- При истечении времени — красные цифры со знаком минуса
- Продолжает считать в минус после истечения

### Звуковые оповещения
- За 1 минуту до конца — воспроизведение предупреждающего звука
- При истечении — воспроизведение финального звука
- Звуковые файлы выбираются пользователем

## Архитектура
- Состояние таймера хранится в Zustand store (Control window)
- Синхронизация с Presentation window через IPC (`timer-update`)
- Оверлей рендерится в PresentationApp как абсолютный элемент
- Звук воспроизводится в Control window через HTML Audio API
